Denne definisjonslista er laga utelukkande for å informere. Ho kan ikkje verte sett på som ei komplett eller offisiell skildring av emna som vert tekne opp, eller dei personvern-/sikkerheitsproblema som vert skildra.

Definisjonar

autentisering
Bruken av passord, sertifikat, personleg identifikasjonsnummer (PIN) eller annan informasjon for å stadfeste ein identifikasjon over eit datanettverk.
bokmerke
Ei lagra adresse (URL) til ei nettside som du enkelt kan gå til ved å trykkje på ikonet til bokmerket i verktøylinja for bokmerke eller velje bokmerket i Bokmerkemenyen.
Verktøylinje for bokmerke
Den lett tilpasselege verktøylinja som ligg rett under adresselinja i &brandShortName; (om du ikkje sjølv har flytta på ho tidlegare). Ho inneheld knappar for bokmerka dine (eller mapper som inneheld grupper med bokmerke) som du kan leggje til eller fjerne.
mellomlager
Ei samling med kopiar av nettsider som er lagra på hardisken eller i minnet (RAM). &brandShortName; tek vare på disse kopiane medan du surfar på Internett. Når du klikkar på ei lenkje eller skriv inn ein URL til ei nettside vil &brandShortName; undersøkje om nettsida finst i mellomlageret. Om nettsida ikkje har endra seg vil &brandShortName; bruke sida i mellomlageret i staden for å laste ho ned på nytt. Dette sparar prossesorkraft og båndbredde.
sertifikat
Det som i den digitale verda tilsvarer ID-kort. Eit sertifikat inneheld namnet til ein person, eit firma eller liknande, og stadfestar at ein fellesnøkkel, som er inkludert i sertifikatet, tilhøyrer denne personen/firmaet/liknande. Når du digitalt signerer ei melding eller andre data, blir ein digital signatur oppretta for denne meldinga med grunnlag i den private nøkkelen, som samsvarer med fellesnøkkelen i sertifikatet ditt.
klient
Programvare (som til dømes ein nettlesar) som sender førespurnader til og mottek informasjon frå ein tenar, som vanligvis køyrer på ei anna datamaskin. Ei datamaskin der klientprogramvare køyrer, vert og kalla ein klient.
infokapsel
Ein liten bit med informasjon som vert lagra på datamaskina di av nokre nettstader. Når du vitjar ein slik nettstad, spør nettstaden &brandShortName; om å setje ein eller fleire infokapslar på harddisken din. Seinere, når du kjem tilbake til denne nettstaden, sender &brandShortName; infokapslane som tilhøyrer nettstaden tilbake. Infokapslar let nettstader vedlikehalde informasjon om deg, som til dømes innhaldet av «handlevogna» di i ein nettbutikk. Bruk av infokapslar i nettlesaren kan styrast gjennom innstillingar som bestemmer korleis infokapslar skal brukast og kor mykje informasjon som kan lagrast i dei.
kryptografi
Kunsten og utføringa av kryptering (eller uleseleggjering) og dekryptering av informasjon. Til dømes vert kryptografiske teknikkar brukte på informasjonen som går mellom kommersielle nettstader og &brandShortName;.
dekryptering
Prosessen å gjere kryptert informasjon leseleg att. Sjå og kryptering.
digital ID
Sjå sertifikat.
kryptering
Prosessen med å gjere data uleselege slik at innhaldet vert skjult for andre. Til dømes, krypterte tilkoplingar mellom datamaskiner gjer det veldig vanskelig for andre å kunne gjere leseleg (dekryptere) informasjonen som flyt over tilkoplinga. Kryptert informasjon kan berre dekrypterast av nokon som har ein passende nøkkel.
Extensible Markup Language (XML)
Ein open standard for å skildre data. I motsetnad til HTML, tillet XML utviklaren av nettsider å lage eigne merkelappar (taggar). For meir informasjon, sjå W3C-dokumentet Extensible Markup Language (XML).
nyhendekjelde (feed)
Ei XML-nettside som inneheld lenkjer til andre nettsider. Spesielle program kan lese XML-nettsida for å presentere lenkjeoverskriftene, og automatisk oppdatere lista når dokumentet vert endra. Nyhendenettsider brukar nyhendekjelder for å raskt publisere siste overskrifter, medan bloggarar og andre personlege journalar på Internett ofte bruker XML-nettsider for raskt å informere vitjande om nye journaloppføringar. Sjå óg levande bokmerke.
File Transfer Protocol (FTP)
Ein standard som gir brukarar sjansen til å overføre filer frå ei datamaskin til ei anna over eit nettverk. Du kan bruka &brandShortName; til å hente filer over FTP-protokollen.
FIPS PUBS 140-1
Federal Information Processing Standards Publications (FIPS PUBS) 140-1 er ein amerikansk standard for implementering av kryptografiske modular — maskinvare eller programvare som krypterer eller dekrypterer data, eller utfører andre kryptografiske operasjonar (som til dømes å kontrollere eller opprette digitale signaturar). Mange produkt som vert selde til ulike regjeringar må støtte ein eller fleire av FIPS-standardane.
startside
Sida som &brandShortName; viser kvar gong du startar han, eller når du trykkjer på Heim-knappen. Omgrepet vert og brukt om hovudsida for ein nettsted, der du kan navigere til andre sider på nettsteden.
Hypertext Markup Language (HTML)
Dokumentformatet som vert brukt på nettsider. HTML-standarden definerer merkelappar, eller kodar, som definerer tekstutsjånad, skrifttypar, stilar, bilete og andre element som nettsider er bygde opp av.
Internett
Eit verdensomspennande nettverk av millionar av datamaskiner som kommuniserer med kvarandre gjennom ein standard protokoll som til dømes TCP/IP. Internett vart opphavleg ferdigutvikla av det amerikanske militæret i 1969, men vaks til også å omfatte forsknings- og utdanningsinstitusjonar. I slutten av 1990-åra vart og fleire millionar bedrifter, organisasjonar og privatpersonar representerte på Internett. I dag bruker vi Internett for sending og mottak av e-post, bruk av verdsveven (World Wide Web), lynmeldingar, e-postlister, nyhendegrupper og mykje anna.
Internet protocol address (IP address)
Adressa til ei datamaskin på eit TCP/IP-nettverk. Kvar datamaskin på Internett har ei eiga IP-adresse. Klientar har enten ei permanent IP-adresse, eller vert tildelte ei ny adresse kvar gong dei kopler seg til nettverket.
Java
Eit programmeringsspråk utvikla av Sun Microsystems. Eit enkelt Java-program kan køyra på mange ulike typar datamaskiner, og unngår slik at programmerarar må lage eigne utgåver av programma sine for kvar type datamaskin. Etter at Java er installert, kan &brandShortName; lasta ned og køyra Java-program automatisk. Java-program som er del av ei heimeside vert kalla applets.
JavaScript
Eit skriptspråk som ofte vert brukt til å lage nettsider. Programmerarar kan bruke JavaScript for å gjere nettsidene meir interaktive. JavaScript vert ofte brukt til å kontrollere data i skjema. JavaScript kan brukast saman med Java, men teknisk sett er det eit heilt eige språk. Ein treng ikkje å ha Java for at JavaScript skal virke skikkeleg.
Levande bokmerke
Ein spesiell type bokmerke som oppfører seg som ei mappe, og inneheld alle lenkjene til eit XML-nyhendedokument. Dersom du opprettar eit levande bokmerke ved å vitje ein nettstad som har eit XML-nyhendedokument, kan du trykkje på ikonet for levande bokmerke i statuslinja, og velje nyhendeformatet du vil bruke.
Adresselinje
Innskrivingsfeltet (og tilhøyrande knappar) nær toppen av &brandShortName;-vindauget der du kan skrive inn ein URL eller eit søkjeord.
hovudpassord
Eit passord som vert brukt til å verne lagra passord og andre private data. &brandShortName; vil spørje etter hovudpassordet når du prøver å få tilgang til desse dataene. Dersom du har fleire sikkerheitseiningar, vil kvar sikkerheitseining krevje eit eige hovudpassord.
Verktøylinje for navigasjon
Ei verktøylinje nær toppen av &brandShortName;-vindauget som inneheld Tilbake og Fram-knappene.
Passordhandsamar
Delen av &brandShortName; som hjelper deg til å hugse alle eller nokre av brukarnamna og passorda du bruker, ved å lagre dei på harddisken på datamaskina, og skrive dei inn i skjema automatisk når du vitjar nettsider der dei trengst.
PKCS #11
Ein fellesnøkkel kryptografistandard som styrer sikkerheitseiningar som til dømes smart-kort.
Programtillegg
Programtillegg legg til nye visningsmoglegheiter i &brandShortName;, som til dømes høve til å spele av lyd- eller videoklipp. I motsetnad til andre typar hjelpeprogram, vert programtillegg lagt til i Plugins-mappa til installeringsmappa til &brandShortName;. Dei kan opnast inne i &brandShortName; sjølv (innbundne i vindauget). Til dømes kan eit lyd-programtillegg late deg høyre på lydfiler på nettsider eller i e-postmeldinger. Macromedia Flash Player og Java er begge døme på programtillegg.
privat nøkkel
Ein av paret av nøklar som vert brukt i fellesnøkkel-kryptografi. Den private nøkkelen er hemmeleg, og vert brukt til å dekryptere data som har vorte krypterte med ein tilsvarande fellesnøkkel.
mellomtenar
Ein tenar eller eit tenarprogram som verkar både som ein tenar og klient med den hensikta å kunne føreta førespurnader på vegne av andre klientar.
søkjemotor
Eit nettsidebasert program som let brukarar søkje etter og hente ut særskild informasjon frå Internett (WWW). Søkjemotoren søkjer etter fleire nøkkelord i den fullstendige teksten til mange nettsider. Nokre søkjemotorar kan og vere baserte på kataloglister som er haldne ved like manuelt. Vanlegvis skriv brukaren inn eit ord eller eit uttrykk i ein søkjeboks, og søkjemotoren vil då vise dei dokumenta som han meiner er relevante til søkjeuttrykket.
sikker nettstad
Ein nettstad som bruker kryptering i tilkoplingar til &brandShortName; til å hindre at ondsinna internettbrukarar kan sjå data som vert overførde. Når du vitjar ein sikker nettstad, vil &brandShortName; vise eit hengelås-ikon i Statuslinja og i adresselinja. &brandShortName; viser og nettstaden sitt domenenamn i statuslinja (for å hindre ondsinna nettstader frå å stele data frå deg) og gjer bakgrunnen til adresselinja gul.
Nokre nettstader kan innehalde både sikre og usikra data. Usikra data er vanlegvis ikkje privat, men ikkje alltid. For slike nettstader vil &brandShortName; vise eit hengelås-ikon med ein strek over i statuslinja og i adresselinja, og vil ikkje vise nettstaden sitt domenenamn i adresselinja, slik at du kan vite at nettstaden ikkje er heilt sikker.
Secure Sockets Layer (SSL)
Ein protokoll som tillet autentisering mellom ein klient og ein tenar for å kunne opprette ei autentisert og kryptert tilkobling. SSL køyrer over TCP/IP-nivået, men under HTTP, LDAP, IMAP, NNTP, og andre nettverksprotokoller på høgre nivå. Den nye Internet Engineering Task Force (IETF)-standarden Transport Layer Security (TLS) er basert på SSL. Sjå og autentisering, kryptering.
tenar
Programvare (slik som programvaren som deler ut nettsider) som mottek førespurnader, og sender informasjon til ein klient, som oftast køyrer på ei anna datamaskin. Ei maskin som har køyrande tenarprogram, kan og kallast ein tenar.
Statuslinje
Ei verktøylinje som visest på botnen av &brandShortName;-vindauget. Ho viser status-ikon på høgre side.
TLS
Sjå Secure Sockets Layer (SSL).
Transmission Control Protocol/Internet Protocol (TCP/IP)
Ein Unix-protokoll som vert brukt til å kople samman datamaskiner som kan bruke vidt forskjellige operativsystem. TCP/IP er ein viktig internett-protokoll og har vorte standard over heile verda .
Uniform Resource Locator (URL/Nettadresse)
Den standariserte adressa som fortel &brandShortName; korleis han finn filer og andre ressursar på Internett (F.eks, http://www.mozilla.org). Skriv URL-en inn i &brandShortName; si adresselinje for å gå til sida du ønskjer å vitje. URL-ar vert óg brukte i lenkjer som peikar til andre sider. URL er på norsk oftast omtala som nettadresse.
nettside
Eit enkelt dokument på WWW (World Wide Web) som er identifisert med ei unik adresse eller URL. Ei nettside kan innehalde tekst, hyperlenkjer og bilete/grafikk.
nettstad
Ei gruppe av nettsider som heng saman med hyperlenkjer og er administrert av eit firma, ein organisasjon eller ein inviduell person. Ein nettstad kan innehalde mange nettsider med tekst, grafikk, lyd og videofiler saman med lenkjer til andre nettsider og nettstader.
Verdsveven (World Wide Web)
Ein stor del av Internett er vev-sider eller web-sider. Desse vert lagra på tenarar og vert framviste med program som vert kalla klientar eller nettlesarar (som til dømes &brandShortName;).
